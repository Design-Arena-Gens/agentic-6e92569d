RC Car Control System

This project contains the complete source code for a Bluetooth-controlled RC car system.

Note: This is a placeholder page. The core of this project is a native Android application and Arduino firmware, which cannot be hosted or run directly on a web server. The code for both components is located in the repository.

Components

  • Android Application: A native Android app built with Kotlin and Jetpack Compose for controlling the car. The source code is in the /AndroidApp directory.
  • Arduino Firmware: The firmware for the Arduino Uno microcontroller that manages the car's motors. The source code is in the /ArduinoFirmware directory.

How to Use

  1. Flash the Firmware:
    • Open the ArduinoFirmware/RcCarFirmware/RcCarFirmware.ino file in the Arduino IDE.
    • Connect your Arduino Uno to your computer.
    • Upload the sketch to the board.
  2. Build and Install the Android App:
    • Open the AndroidApp project in Android Studio.
    • Build the project and install the APK on your Android device.
  3. Connect and Control:
    • Ensure the RC car is powered on.
    • Pair your Android phone with the "HC-05" Bluetooth module in your phone's Bluetooth settings.
    • Open the "RC Car" app. It should automatically connect to the car.
    • Use the on-screen controls to drive the car.
